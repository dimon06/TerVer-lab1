# Лабораторная работа по теории вероятности №1

Выполнял - Хасанов Дмитрий Вадимович M3239

Хасанов Дмитрий. Х, А, Д, М — 23, 1, 5 и 14 буквы в кириллице соответственно. Номера вариантов:

    — (23 % 2) + 1 = 2
    — (5 % 3) + 1 = 3
    — (1 % 3) + 1 = 2
    — (14 % 3) + 1 = 3
    — (2 + 3 + 2 + 3) % 5 + 1 = 1
    — (5 % 2) + 1 = 2.
